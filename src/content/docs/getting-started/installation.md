Instructions for installing the Neo compiler will be available here once the compiler reaches its first public release.

## Building from Source

For now, you can build Neo from source:

```bash
git clone https://codeberg.org/neolang/neo.git
cd neo
make
```

## Next Steps

Once installed, follow the [Hello World](/getting-started/hello-world/) tutorial to write your first Neo program.
